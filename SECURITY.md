# Security Policy

## Supported Versions

Security fixes are provided for the latest release only.

| Version | Supported |
| ------- | --------- |
| Latest release | ✅ |
| Older releases | ❌ |

Check the latest version on:
- [GitHub Releases](https://github.com/quackawop-rgb/Geasd/releases)
- [Modrinth](https://modrinth.com/mod/geasd)

---

## Reporting a Vulnerability

**Do not report security vulnerabilities through public GitHub Issues or Modrinth comments.**

If you believe you have found a security issue in Geasd, please report it privately:

1. Go to the [Geasd repository](https://github.com/quackawop-rgb/Geasd)
2. Open **Security** → **Report a vulnerability** (Private vulnerability reporting)

If private reporting is unavailable, contact the repository owner directly through GitHub and avoid posting details publicly on GitHub, Modrinth, or other platforms.

---

## What to Include

Please include as much detail as possible:

- Description of the issue
- Steps to reproduce
- Geasd version and file hash (if possible)
- Where you downloaded the mod (GitHub Releases or Modrinth)
- Minecraft version, Fabric Loader, and Fabric API versions
- Whether the issue affects singleplayer, multiplayer, or both
- Impact assessment (what an attacker could do)
- Any proof-of-concept, logs, or screenshots (keep PoCs minimal and safe)

---

## In Scope

Examples of security issues we take seriously:

- Malicious code execution through the mod
- Arbitrary file read/write outside intended game/mod behavior
- Server-side exploits that affect other players without consent
- Supply-chain concerns (tampered JARs, compromised release artifacts)
- Credential or sensitive data exposure caused by the mod
- Denial-of-service issues that crash servers or clients reliably through mod behavior

---

## Out of Scope

Please use [GitHub Issues](https://github.com/quackawop-rgb/Geasd/issues) for non-security reports, such as:

- Gameplay bugs (wrong seed drops, stubble not regrowing, etc.)
- Visual glitches or missing textures
- Compatibility problems with other mods
- Feature requests
- Crashes without a plausible security impact

Modrinth comments and reviews are not monitored for bug reports — use GitHub Issues instead.

---

## Response Timeline

| Stage | Target |
| ----- | ------ |
| Initial acknowledgment | Within **7 days** |
| Status update | Within **30 days** |
| Fix or mitigation plan | Depends on severity |

Timelines may vary for complex issues. We will keep you updated when possible.

---

## Safe Harbor

We appreciate responsible disclosure. If you:

- Report in good faith
- Avoid privacy violations, data destruction, or disruption to other users
- Do not exploit the issue beyond what is needed to demonstrate it

…we will not pursue action against you for that research.

## Official Downloads

Only download Geasd from these official sources:

- [GitHub Releases](https://github.com/quackawop-rgb/Geasd/releases)
- [Modrinth](https://modrinth.com/mod/geasd)

Do not trust reuploads from unofficial sites, third-party mirrors, or unknown file hosts. Verify you are using the official GitHub repository or Modrinth project page.
