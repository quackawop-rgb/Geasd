# Contributing to Geasd

Thanks for helping improve Geasd!

## How to Help

Geasd is maintained by the project owner. Your feedback is very **welcome.**

### Bug reports — encouraged
Found something broken? **[Open an issue](https://github.com/quackawop-rgb/Geasd/issues)**.

Please include:
- Minecraft, Fabric Loader, and Fabric API versions
- Geasd version
- Singleplayer or multiplayer
- Steps to reproduce
- Logs or crash reports if you have them

### Feature & change requests — encouraged
Have an idea for how Geasd should work? **[Open an issue](https://github.com/quackawop-rgb/Geasd/issues)** and describe what you’d like changed and why.

There’s no guarantee every request will be added, but all suggestions are read and considered.

### Security issues
Do **not** report security problems in public issues. See [SECURITY.md](SECURITY.md).

---

## Pull requests

**Pull requests are not accepted** at this time. All code changes are made by the project owner.

If you want something changed, please open an issue instead — that’s the best way to request it.
